Профессиональная утилита для очистки, конвертации и визуализации гидравлических моделей **EPANET**. 
Проект построен по классической архитектуре **ETL (Extract, Transform, Load)** и способен напрямую парсить проприетарные бинарные файлы Delphi.

##  Особенности проекта

*   **Нативный парсинг `.NET` файлов**: Программа читает бинарные файлы EPANET напрямую.
*   **Поддержка `.epanet` архивов**: Автоматическая распаковка и чтение ZIP-контейнеров современных версий ПО.
*   **Умная очистка**: Удаление лишних графических секций (BACKDROP, LABELS, TAGS), пустых строк и комментариев для уменьшения веса модели.
*   **Универсальный экспорт**: Сохранение очищенной модели в классический текстовый `.INP`, бинарный `.NET` (в текстовом представлении) или запакованный `.EPANET` архив.
*   **Встроенный Viewer**: Интерактивная отрисовка гидравлического графа (узлы, резервуары, ёмкости, насосы, задвижки, трубы) с поддержкой Pan & Zoom прямо в приложении.

##  Архитектура (ETL)

Проект строго разделен на независимые модули:

*   📂 `src/extract/` — **Чтение данных**. Включает фабрику парсеров (`factory.py`), текстовый парсер (`inp_parser.py`) и бинарный парсер (`net_parser.py`).
*   📂 `src/transform/` — **Трансформация**. Очистка от мусора и комментариев (`cleaner.py`).
*   📂 `src/load/` — **Сохранение**. Формирование структуры и упаковка данных в целевые форматы (`writer.py`).
*   📂 `src/ui/` — **Интерфейс**. GUI на основе `tkinter` и интерактивный холст для графов.
